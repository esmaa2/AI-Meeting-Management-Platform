# MeetlyAI Frontend

Frontend application for MeetlyAI built with Angular.

---

## Features

### Authentication Pages
- Login
- Registration
- Password Recovery

### Dashboard
- Productivity Metrics
- Meetings Overview
- Upcoming Priorities

### Meeting Management
- Create Meeting
- Transcript Submission

### AI Assistant
- Interactive AI Chat Interface
- Productivity Recommendations

### User Settings
- Profile Management
- Notifications
- Integrations
- Billing

---

## Technologies

- Angular
- TypeScript
- HTML
- CSS

---

## Installation

npm install

npm start

---

## Build

npm run build
